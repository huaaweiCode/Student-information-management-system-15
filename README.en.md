# 学生信息管理系统15

#### Description
学生信息管理系统15拥有三种角色

- 管理员：老师管理、学生管理、留言板/教学资料/授课管理、通知管理等
- 教师：班级人员信息查询、授课安排、上传教学资料、查看回复学生留言、查看通知等
- 学生：查看下载学生资料、查看授课安排、查看通知等
